# architecture-pro-propdevelopment

Проектная работа по курсу «Архитектура». Кейс компании PropDevelopment - крупного застройщика, который готовится к комплексному аудиту безопасности.

## О кейсе

PropDevelopment входит в топ-3 застройщиков страны и предоставляет IT-продукты для двух категорий пользователей:

- **Клиенты (потенциальные покупатели)** - онлайн-витрина, онлайн-тур, онлайн-сделка
- **Собственники (купившие жильё)** - мобильное приложение для ЖКУ, умный дом, информация от УК

Ключевые проблемы компании: нарушен контроль данных и небезопасная интеграция с API партнёров (управляющих компаний).
